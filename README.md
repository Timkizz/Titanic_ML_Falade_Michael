# Project Overview 

This project is an exploratory analysis of the Titanic dataset, completed as part of the AnalystLab Africa internship assessment.  

The goal is to perform data preprocessing and Exploratory Data Analysis on the Titanic Dataset 

 

### Methodology and Process 

Data Cleaning: Dropped Unnecessary columns and handled missing values in the Age and Embarked columns to ensure data integrity 

Exploratory Data Analysis (EDA): Visualized survival distributions across various features such as Gender, Passenger Class, Age 

Feature Encoding: Converted categorical data (Sex, Embarked) into numerical columns 

Data Normalization:  Used StandardScaler to scale the Age and Fare columns 

 

### Key insights 

Gender : Females had a significantly higher survival rate (correlation : 0.54) 

Socio-economic Class : 1st class passengers were prioritized for lifeboats and had the highest survival percentage 

Age: Younger passengers (children) showed higher survival rates in certain classes indicating a “women and children first” protocol

Language: Python
